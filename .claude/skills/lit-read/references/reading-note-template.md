# Reading Note Template

Use this template when extracting structured notes from individual papers in Phase 2 (Close Reading).

---

## Per-Paper Note Format

```markdown
# [Full Title]

**Authors**: [Author1, Author2, ...]
**Year**: [YYYY]
**Journal**: [Journal Name] (ABS: [rating])
**DOI**: [doi string]
**Thread**: [Thread name from research-map.md]
**Role**: [origin | extension | critique | turning-point | synthesis]

---

## Core Argument

[1-2 sentences: What is this paper's central claim or thesis?]

## Research Questions

1. [RQ1]
2. [RQ2]
...

## Key Findings

- [Finding 1 with effect size / significance if available]
- [Finding 2]
- [Finding 3]

## Method

- **Design**: [experimental / longitudinal / cross-sectional / mixed / qualitative]
- **Sample**: [N, population, context]
- **Analysis**: [statistical method / qualitative approach]
- **Key variables**: [main IV, DV, mediators, moderators]

## Limitations

**Acknowledged by authors:**
- [Limitation 1]

**Not acknowledged (reviewer perspective):**
- [Limitation 1]

## Connection to This Project

**Relevant thesis part(s):** [Part 1 / Part 2 / Part 3]

**How it connects:**
[1-3 sentences on direct relevance to this project]

## Key Concepts to Extract

- [Concept/construct name]: [brief definition]
- [Theoretical mechanism]: [how it works]
- [Methodological approach]: [technique worth noting]
- [Empirical evidence]: [finding to cite]

## Knowledge Framework Format

```markdown

**Papers analyzed**: N | **Threads**: K | **Date**: YYYY-MM-DD

## Theoretical Landscape

### [Theory/Framework Name]
- **Core idea**: [1-2 sentences]
- **Key proponents**: [Author (Year), Author (Year)]
- **Empirical support**: [which papers, what evidence]
- **Critiques**: [which papers, what challenges]
- **Relevance to This Project**: [connection to project]

### [Another Theory/Framework]
...

## Methodological Patterns

| Method | Papers Using It | Strengths | Weaknesses |
|--------|----------------|-----------|------------|
| [Method] | [Paper1, Paper2] | ... | ... |

## Empirical Evidence Map

### Evidence for [Construct/Mechanism A]
- **Strong**: [Paper (Year)] — [finding]
- **Mixed**: [Paper (Year)] — [finding]
- **Weak/Contradicted**: [Paper (Year)] — [finding]

### Evidence for [Construct/Mechanism B]
...

## Open Debates & Unresolved Tensions

1. **[Debate name]**: [Paper X says A, Paper Y says B — tension is...]
2. **[Debate name]**: ...

## Research Gaps for This Project
