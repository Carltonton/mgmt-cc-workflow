# Critical Evaluation & Writing Guide

## Critical Evaluation Framework

For each key paper, evaluate along three dimensions:

### 1. Theoretical Contribution
- Does it extend existing theory?
- Does it introduce new constructs?
- Does it resolve contradictions?

### 2. Methodological Quality
- Is the design appropriate?
- Are measures validated?
- Is sample adequate?
- Are analyses appropriate?

### 3. Dialogue Contribution
- Does it challenge conventional wisdom?
- Does it provide counter-intuitive insights?
- Does it bridge disconnected conversations?

---

## Building Theoretical Conversations

### Structure of a Literature Review

1. **Introduction**
   - State purpose and scope
   - Define key concepts
   - Preview organization

2. **Body Paragraphs** (Organized by theme or chronology)
   - Each paragraph has a clear topic sentence
   - Synthesize, don't just list
   - Show connections between studies
   - Identify tensions and contradictions

3. **Gap Identification**
   - Explicitly state what's missing
   - Explain why the gap matters
   - Preview how your study addresses it

4. **Conclusion**
   - Summarize key points
   - Transition to your study

### Transition Words for Synthesis

| Purpose | Transitions |
|---------|-------------|
| **Adding** | Additionally, Furthermore, Moreover, Also |
| **Contrasting** | However, In contrast, Conversely, Nevertheless |
| **Sequencing** | First, Second, Third, Finally, Subsequently |
| **Causality** | Therefore, Thus, Consequently, As a result |
| **Similarity** | Similarly, Likewise, In the same vein |
| **Comparison** | Compared to, In comparison, Whereas |

## Common Pitfalls to Avoid

| Pitfall | Problem | Solution |
|---------|---------|----------|
| **List-based review** | Just summarizing papers | Synthesize across papers |
| **Annotated bibliography** | No integration | Build narrative flow |
| **Unfocused** | Too broad, no clear purpose | Define scope clearly |
| **Missing key papers** | Incomplete coverage | Comprehensive search, expert review |
| **No gap identified** | No rationale for new study | Explicitly identify what's missing |
| **Outdated sources** | No recent work | Include recent research |
